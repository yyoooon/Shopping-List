# ShoppingList
javascript를 사용해 TodoList어플리케이션을 만들었습니다.  

**구현 기능 목록**
- 리스트 추가
- 완료된 리스트 표시
- 리스트 선택 삭제
- 리스트 전체 삭제

## Usage
- 폴더를 다운 받고 **index.html**을 실행시킵니다.
- 추가된 리스트를 한번 더 누르면 완료 표시가 생깁니다.
